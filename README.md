## About This Site

This is a single-page website built for family recruitment and community outreach. It includes:

- Overview of the Flourish model and philosophy
- Daily schedule walkthrough
- Pursuits (project-based learning) examples
- SuperPowers skills framework
- Tuition & Arizona ESA funding information
- Founding cohort details
- Parent interest / signup form (connected to Formspree)

---

## About Flourish Microschool

Flourish is a private independent microschool network founded by **John Danner** (co-founder of Rocketship Public Schools) and **Adam Nadeau**. Flourish schools are:

- Intentionally small — max 25 students per teacher
- Skills-first — 24 SuperPowers tracked daily with AI
- Project-based — afternoons protected for student-led Pursuits
- Accessible — Arizona ESA funding may cover most or all of tuition
- MSA Accredited
